Avant-Garde · Case Study

Avant-Garde is a fictitious art supplies store competing against DeSerres. This Broadcast Package project was initially a Branding assignment from the k6 Media design agency. The logo was inspired by the visual arts and design, more specifically, artisan tools like paint swatches.
